# 🚀 Najwa LEGHRIS — Portfolio React + TypeScript

Portfolio professionnel avec animations avancées.

## Stack
- **React 18** + **TypeScript**
- **Framer Motion** — animations & transitions de pages
- **tsParticles** — particules animées en fond
- **Vite** — bundler ultra-rapide

## Animations incluses
- ✅ Page transition (overlay au chargement)
- ✅ Particules interactives en fond
- ✅ Terminal avec typewriter effect
- ✅ Parallaxe sur le hero (scroll)
- ✅ Scroll reveal sur toutes les sections
- ✅ Grille hexagonale 3D interactive
- ✅ Tilt 3D sur les cartes expérience
- ✅ Radar chart animé
- ✅ Dark/Light mode (toggle + auto-détection OS)
- ✅ Curseur personnalisé

## Installation

```bash
# 1. Installer les dépendances
npm install

# 2. Lancer en développement
npm run dev

# 3. Build pour production
npm run build
```

Ouvre http://localhost:5173 dans ton navigateur.

## Personnalisation

Toutes les données sont dans `src/data/index.ts` :
- Modifie tes skills, expériences, formations
- Tout se met à jour automatiquement

## Déploiement (Vercel / Netlify)

```bash
npm run build
# Upload le dossier /dist sur Vercel ou Netlify
```
